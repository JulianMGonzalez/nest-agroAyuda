## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start:dev

# seed
$ http://localhost:3000/api/seed

# docker database
$ docker-compose up -d
```
